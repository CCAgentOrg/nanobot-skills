# nanobot-skills

Custom skills for nanobot AI assistant.

## Skills

### digest-tracker
Track topics, fetch news from media/blog sources, and generate digests for WhatsApp or blog posts. Supports Turso storage, RSS feeds, and blog publishing to Jekyll/Hugo.

### github-watcher
Track and monitor GitHub repositories with latest release, version number, and star count. Fetch and summarize release notes.

### youtube-recommender
Recommends relevant YouTube videos for any topic, with duration filtering and engagement-based ranking.

### zai-quota
Check your Z.AI GLM Coding Plan usage statistics with real-time quota monitoring, plan tier detection, and token usage tracking.

### xpost
Draft engaging X (Twitter) posts and threads with optimal structure, formatting, and engagement elements. Includes templates, hooks library, and real examples.

### x-poster
Post tweets to X (Twitter) from nanobot. Supports text tweets, images, threads, replies, and tweet deletion using the X API v2.

### x-gif-maker
Create animated GIFs and MP4s for social media. Generate text-based animations, multi-headline slideshows, and branded content with auto-upload to catbox.moe.

## Installation

Copy skill directories to `/home/ubuntu/.nanobot/workspace/skills/` and restart nanobot.
