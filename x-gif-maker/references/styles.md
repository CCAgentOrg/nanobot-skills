# Animation Styles

Available animation styles for creating engaging social media GIFs.

## Basic Styles

### Fade
**Best for:** Introductions, announcements, clean minimalist posts
**Effect:** Text smoothly fades in from transparent to fully visible
**Duration:** 2 seconds recommended
**Use cases:**
- Product announcements
- Quote posts
- Feature highlights
- Welcome messages

```
style="fade"
```

### Pulse
**Best for:** Call-to-actions, important updates, grabbing attention
**Effect:** Text gently pulses in brightness/intensity
**Duration:** 2-3 seconds recommended
**Use cases:**
- "New feature available!"
- "Breaking news:"
- "Don't miss out!"
- Limited time offers

```
style="pulse"
```

### Slide Up
**Best for:** Rising news, positive announcements, uplifting content
**Effect:** Text slides up from bottom while fading in
**Duration:** 2 seconds recommended
**Use cases:**
- Success stories
- Achievement highlights
- Growth metrics
- Positive updates

```
style="slide_up"
```

### Slide Down
**Best for:** Announcements, reveals, dropping news
**Effect:** Text slides down from top while fading in
**Duration:** 2 seconds recommended
**Use cases:**
- "Introducing..."
- "Coming soon..."
- "Dropping now:"
- New releases

```
style="slide_down"
```

### Bounce
**Best for:** Fun, energetic content, playful announcements
**Effect:** Text bounces in with an elastic ease effect
**Duration:** 2 seconds recommended
**Use cases:**
- Fun updates
- Celebration posts
- Interactive content
- Casual announcements

```
style="bounce"
```

## Advanced Styles

### Glow
**Best for:** Premium content, feature highlights, luxury branding
**Effect:** Text fades in with a glowing outer effect
**Duration:** 2-3 seconds recommended
**Use cases:**
- Premium features
- Special announcements
- Brand reveals
- Highlight text

```
style="glow"
```

### Shake
**Best for:** Urgent alerts, breaking news, error messages
**Effect:** Text shakes horizontally as it appears
**Duration:** 2 seconds recommended
**Use cases:**
- "Breaking:"
- "Important:"
- Error/fix messages
- Critical updates

```
style="shake"
```

### Typewriter
**Best for:** Code snippets, developer content, progressive reveals
**Effect:** Text appears character by character
**Duration:** 3 seconds recommended for longer text
**Use cases:**
- Code announcements
- API endpoints
- Developer updates
- Teaser text

```
style="typewriter"
```

### Zoom
**Best for:** Impact reveals, major announcements, punchy text
**Effect:** Text scales up from 50% to 100% while fading in
**Duration:** 2 seconds recommended
**Use cases:**
- "MAJOR UPDATE"
- Version releases
- Bold announcements
- Impact headlines

```
style="zoom"
```

### Wave
**Best for:** Casual, flowing content, organic feel
**Effect:** Text gently waves up and down
**Duration:** 2 seconds recommended
**Use cases:**
- Casual updates
- Organic feel
- Friendly messages
- Lighthearted content

```
style="wave"
```

### Glitch
**Best for:** Tech announcements, cybersecurity, edgy content
**Effect:** RGB split glitch effect with horizontal shifts
**Duration:** 2 seconds recommended
**Use cases:**
- Tech releases
- Cybersecurity
- Edgy branding
- Digital native content

```
style="glitch"
```

## Size Guidelines

### X (Twitter) Optimized
- Width: 640px
- Height: 360px (16:9)
- Duration: 1-3 seconds
- File size: <5MB preferred, max 15MB

### X (Twitter) Premium
- Width: 1280px
- Height: 720px (16:9)
- Duration: 1-3 seconds
- File size: <10MB preferred

### Instagram Stories
- Width: 640px
- Height: 1136px (9:16)
- Duration: 2-4 seconds
- File size: <4MB

### Instagram Feed

- Duration: 2-4 seconds
- File size: <4MB
